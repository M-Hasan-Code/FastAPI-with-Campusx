# HR Analytics - FastAPI ML Project

A FastAPI-based project that demonstrates:

* 🚀 **CRUD Operations** (Create, Read, Update, Delete) for Patient data
* 🧠 **ML Model Serving** – HR Analytics prediction
* 📊 **End-to-End Workflow** – from data preprocessing to deployment
* 📑 **Interactive API Docs** – Swagger & ReDoc

## 🛠 Tech Stack

* **FastAPI**, **Uvicorn**, **Pydantic**
* **scikit-learn**, **pandas**, **numpy**
